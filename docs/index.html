The Minto pattern

The site and the answer follow the same logic.

A professional document assistant should not make the reader hunt. It should present the conclusion, then the argument, then the proof.

Answer What is true?
Support Why is it true?
Evidence Where is it proven?
01

Start with the answer

Every response opens with one governing thought, written as a decision-ready sentence.

02

Group the support

Supporting points are compact, parallel, and easy to scan before the reader goes deeper.

03

Attach the proof

Citations stay visible, so every synthesized claim can be traced back to a source page.

Why it feels professional

Less decoration. More judgment.
